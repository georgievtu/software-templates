# Software templates

Software templates lab project for TU Sofia. Demonstrates various design patterns. Examples are intentionally unfinished for students to complete them.

## Syllabus
1. Observer, Template and Factory patterns
2. Facade, Iterator and Proxy patterns
3. Adapter, Composite, Singleton, State, Strategy and Decorator patterns
4. Command, Visitor and Chain of responsibility patterns
5. Project defense

## Requirements

- JDK 1.8

## Download

1. Create a folder.
2. Open a terminal in your folder.
3. Run:
    ```sh
    git clone https://github.com/georgievtu/software-templates
    ```

## Example project

Includes an [example](https://github.com/Alex-Tsvetanov/MetroSim) project by [Alex Tsvetanov](https://github.com/Alex-Tsvetanov) with documentation. Clone recursively to include it as a submodule.
